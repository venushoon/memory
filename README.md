# 🎮 메모리 챌린지 (Memory Challenge Game)

> 전자칠판용 터치형 교육 게임  
> 같은 그림 찾기를 통한 집중력과 기억력 향상

---

## 📌 게임 개요

**메모리 챌린지**는 전자칠판이나 태블릿에서 사용하는 교육용 메모리 게임입니다. 사용자가 업로드한 이미지를 이용해 같은 쌍을 맞추는 게임으로, 학생들의 집중력과 기억력을 재미있게 향상시킬 수 있습니다.

### 🎯 목적
- 학생들의 기억력 및 집중력 향상
- 재미있는 상호작용식 학습 경험 제공
- 개인 및 단체 경쟁 활성화

---

## ✨ 주요 특징

### 1. **완전한 커스터마이징**
- 사용자가 직접 이미지 업로드 가능
- 각 이미지에 한글/영문 라벨 추가 가능
- 최대 16개 이미지까지 지원

### 2. **다양한 게임 모드**
- **단일 플레이**: 개인의 최고 기록 추구
- **대결 모드**: 2명이 번갈아가며 진행 (점수 경쟁)
- **텍스트-이미지 매칭**: 그림 카드와 텍스트 카드를 짝짓는 고급 모드

### 3. **정확한 난이도 조절**
- 8장 카드 (4×2 배열) - 4개 이미지 페어
- 12장 카드 (4×3 배열) - 6개 이미지 페어
- 16장 카드 (4×4 배열) - 8개 이미지 페어

### 4. **게임 환경 설정**
- 🌙 다크 모드 / ☀️ 라이트 모드 전환
- 🔊 효과음 ON/OFF 설정
- ⏱️ 시간 제한 선택 가능 (없음/2분/3분/5분)

### 5. **몰입감 있는 UI/UX**
- 글래스모피즘 디자인
- 매칭 성공 시 파티클 애니메이션 ✨
- 효과음 피드백 (플립음, 정답음, 오류음, 승리음)

### 6. **게임 지원 기능**
- **3-2-1 카운트다운**: 게임 시작 전 준비 시간
- **힌트 시스템**: 3회 제한 (남은 카드 중 랜덤 선택)
- **일시정지**: 선생님 설명 시간 확보
- **진행도 바**: 실시간 게임 진행 상황 표시

### 7. **개인화 및 통계**
- 게임 시작 시 닉네임 입력 (단일 플레이)
- 게임 기록 자동 저장
- 명예의 전당 (TOP 5 리더보드)
- 전체 통계 (총 플레이 수, 최고 기록, 평균 시간)

## 🎮 게임 기능

### 카드 시스템
| 요소 | 설명 |
|------|------|
| 카드 앞면 | 파란색 배경에 "?" 표시 |
| 카드 뒷면 | 업로드된 이미지 또는 텍스트 |
| 매칭 성공 | 카드 투명해짐 + 파티클 효과 |
| 매칭 실패 | 카드 다시 뒤집힘 (1초 후) |

### 게임 진행 규칙
1. 카드를 터치해서 앞뒤를 뒤집기
2. 같은 쌍을 찾으면 카드 제거
3. 모든 카드를 매칭하면 게임 완료
4. 대결 모드: 매칭 성공 시 다음 턴 (점수 +1)
5. 매칭 실패 시 상대방 차례로 변경

### 특수 기능

#### 💡 힌트 시스템
- 3회 제한
- 사용 시 아직 안 맞춘 카드 중 하나가 플래시
- 게임 중 언제든 사용 가능

#### ⏸️ 일시정지
- 게임 진행 중 일시정지 가능
- "선생님의 설명을 들어보세요" 메시지 표시
- 계속하기 버튼으로 재개

#### ⏱️ 시간 제한 모드
- 선택 가능: 없음 / 2분 / 3분 / 5분
- 시간 남음: 상단에 표시
- 30초 남았을 때: 경고음 + 카운트다운

## 📱 사용 방법

### 초기 설정

#### 1️⃣ 홈 화면
```
메모리 챌린지
├─ 🚀 게임 시작하기
├─ 🏆 최근 명예의 전당 (토글식)
└─ 📊 전체 통계 보기
```

#### 2️⃣ 이미지 설정 (사진 설정)
```
🖼️ 사진 설정
├─ 클릭하여 사진 선택 (최대 16개)
├─ 각 사진에 단어/라벨 입력
└─ [다음 단계]
```

**꿀팁**: 사진을 드래그해서 제거 가능, 라벨은 선택사항

#### 3️⃣ 게임 규칙 설정 (게임 설정)
```
⚙️ 게임 설정
├─ 🎴 난이도 선택 (8/12/16장)
│  └─ 📝 텍스트-이미지 매칭 체크
├─ 👥 게임 모드 (한 명/대결)
│  └─ 대결 모드 시 학생 이름 입력
├─ ⏱️ 시간 제한 (선택)
├─ ⚙️ 환경 설정
│  ├─ ☀️ 라이트 모드
│  └─ 🔊 효과음
└─ [🚀 게임 시작!]
```

#### 4️⃣ 닉네임 입력 (단일 플레이만)
```
👤 닉네임을 입력하세요
├─ 엔터 또는 [시작!] 버튼
└─ → 게임 진행
```

**주의**: 대결 모드는 닉네임 입력 없이 바로 시작됨

#### 5️⃣ 게임 준비 카운트다운
```
준비하세요!
3️⃣ (1초)
2️⃣ (1초)
1️⃣ (1초)
시작! → 게임 진행
```

### 게임 진행

#### 게임 화면 레이아웃
```
[좌측 패널]                    [우측 게임판]
├─ 📊 게임 정보                ├─ 4×2 / 4×3 / 4×4 카드
├─ ⏸️ 일시정지                └─ 카드 터치로 게임 진행
├─ ⏱️ 경과 시간
├─ 📊 진행도 바
├─ 💡 힌트 (3회)
├─ 👥 플레이어 점수 (대결만)
└─ 🔄 설정으로
```

#### 카드 매칭 프로세스
1. **카드 터치** → 플립 효과음 🔊
2. **카드 뒤집기** → 600ms 애니메이션
3. **매칭 판단** → 즉시 판정
4. **성공 시** → 파티클 효과 ✨ + 정답음 🎵
5. **실패 시** → 1초 대기 후 뒤집히고, 차례 변경

### 게임 완료

#### 결과 화면
```
🎉 완료!
├─ [개인전] 기록: XX초
├─ [대결전] 승자: 플레이어 이름 🏆
└─ 버튼
   ├─ [다시 하기] → 같은 설정으로 새 게임
   ├─ [설정 바꾸기] → 게임 설정 화면으로
   └─ [홈으로] → 홈 화면으로 돌아가기
```

## 🎯 주요 개선 사항

### 개발 버전 히스토리
```
v1.0 - 기본 게임 로직 구현
v2.0 - 음성 안내, 힌트, 대결 모드 추가
v3.0 - 난이도 자동 선택, 닉네임 시스템 추가
v4.0 - 대결 모드 닉네임 스킵, 결과 화면 타이밍 수정
v5.0 - 카드 배열 최적화 (4×2, 4×3, 4×4)
v6.0 - 완성본 (모든 버그 수정, 기능 100% 구현)
```

### 마지막 수정사항 (v6.0)
✅ 결과 화면 표시 안 됨 → useEffect 의존성 최소화로 해결  
✅ 대결 모드 닉네임 중복 → 모드 별 분기 처리로 해결  
✅ 다시하기 버튼 미작동 → playAgain() 함수로 해결  
✅ 카드 배열 불규칙 → CSS Grid 고정 레이아웃으로 해결  
✅ 정답 효과 타이밍 → 애니메이션 완료 후 1200ms 지연 추가

3. 사진 4개 이상 업로드

### 자주 묻는 질문 (FAQ)

**Q. 사진을 꼭 업로드해야 하나요?**  
A. 네, 최소 4개 이상의 사진이 필요합니다.

**Q. 라벨을 입력하지 않으면?**  
A. 라벨은 선택사항이며, 입력하지 않으면 빈 칸으로 처리됩니다.

**Q. 게임 기록이 어디에 저장되나요?**  
A. 브라우저의 localStorage에 저장됩니다 (최대 50개).

**Q. 데이터를 다른 기기로 옮길 수 있나요?**  
A. 현재는 기기 간 동기화가 지원되지 않습니다.

**Q. 오프라인에서도 사용 가능한가요?**  
A. 네, 완전 오프라인 실행 가능합니다.
